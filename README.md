# React Boiler Plate

Yet another one out of the millions of react-redux-boilerplates that exists out there.

## Usage

- Install dependencies

`yarn install`

- Build File

`yarn build`

- Start Server

`yarn start`

## What's in this boilerplate?

### FrontEnd
- React

### BackEnd
- Express.js

### State Handling
- Redux

### Style
- Radium
- PostCSS
(You don't need both, I know)

### Testing
- Jest
- Enzyme
